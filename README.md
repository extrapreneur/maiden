## Usage

```sh
npm run serve
```

## Build & Deploy

1. Build static files

   ```sh
   netlify build
   ```

2. Deploy to [maiden.ga](https://maiden.ga)

   ```sh
   netlify deploy --prod --message "My awesome build"
   ```
